# Trevo — Privacy Policy

Public privacy policy page for the Trevo mobile app.

Hosted via GitHub Pages at: **https://edencha17-hash.github.io/trevo-privacy/**

## Purpose

This page satisfies Google Play's requirement for a public privacy policy URL, including disclosure of background location usage.

## Deployment

This site is deployed automatically via **GitHub Pages** from the `main` branch (root `/`).

### To update

1. Edit `index.html`
2. Commit and push to `main`
3. GitHub Pages will redeploy automatically (usually within 1-2 minutes)

### Initial setup (already done)

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Source: **Deploy from a branch**
4. Branch: `main` / `/ (root)`
5. Save
