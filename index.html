Privacy Policy

Effective Date: March 18, 2026

Trevo ("we", "our", or "us") operates the Trevo mobile application. This page informs you of our policies regarding the collection, use, and disclosure of personal information when you use our Service.

Information We Collect

1. Location Data (Foreground & Background)

Trevo collects real-time location data to provide its core functionality of live location sharing within user-created groups.

Location data may be collected:

  • While the app is in use (foreground)
  • While the app is running in the background (not when fully closed), only if the user has explicitly enabled location sharing

Background location is used solely to allow continuous location sharing within a group, even when the app is not actively open.

Location tracking is:

  • Fully user-controlled
  • Disabled by default
  • Activated only when the user chooses to share location within a group

Location data is only visible to members of the specific group and is never public.

Users can stop sharing their location at any time via the app settings.

2. Account Information

We may collect basic account information such as username, email address, or phone number.

3. Device Information

We may collect non-personal data such as:

  • Device type
  • Operating system version
  • App usage data

How We Use Information

We use collected data to:

  • Enable real-time group location sharing
  • Allow users to coordinate with group members
  • Improve app performance and reliability
  • Ensure security and prevent abuse

Background Location Justification

Trevo uses background location strictly to support its core feature: continuous group-based location sharing.

This feature allows users to share their live location with selected group members even when the app is running in the background.

Background location is not used for advertising, analytics tracking, or any unrelated purposes.

Users must explicitly enable this feature, and they can disable it at any time.

Data Sharing

We do not sell or rent personal data.

Location data is shared only with group members selected by the user.

Data Security

We implement reasonable security measures, but no system is completely secure.

Data Retention

Data is retained only as long as necessary to provide the service.

Children's Privacy

Trevo is not intended for users under 13.

User Rights

Users may request deletion of their data at any time.
